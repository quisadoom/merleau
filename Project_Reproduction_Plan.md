# 🚗 모리스 메를로-퐁티 연구 대시보드 - 프로젝트 요약 및 실행 가이드

이 문서는 사용자가 프로젝트 디렉토리(`c:\dashboard`)를 USB나 클라우드를 통해 집으로 가져가서 **오류 없이 완전히 똑같이 구동**할 수 있도록 작성된 재현(Reproduction) 매뉴얼 및 구현 요약본입니다.

## 1. 프로젝트 구조 및 요약 (Project Structure)
현재 작업 폴더(`c:\dashboard`) 내에 다음 파일들이 정상적으로 위치해야 합니다.
- `app.py` : 대시보드의 데이터 전처리 로직 및 UI가 정의된 메인 실행 코드.
- `퐁티_1_300.xls` & `퐁티_301_600.xls` : KCI 엔진 기반 엑셀 원본 데이터 (xlrd 호환 방식).
- `dashboard_ex1.md` : 프로젝트 초기 스펙 문서.

## 2. 🏠 집에서 실행하기 위한 환경 설정 (Setup from Home)
집에 있는 PC(Windows)에서 이 대시보드를 띄우기 위해서는 다음 단계를 순서대로 진행하시면 됩니다.

### Step 1: Python 설치 확인
집 PC에 Python 3.9 이상 버전이 설치되어 있어야 합니다. 명령 프롬프트(CMD)나 PowerShell 창을 열어 확인합니다.
```bash
python --version
```

### Step 2: 필요 패키지(모듈) 설치
`app.py`를 실행하기 위한 필수 라이브러리 목록을 설치해야 합니다. 터미널(PowerShell)에서 프로젝트가 있는 폴더로 이동(`cd c:\dashboard`)한 뒤 아래 명령어를 입력하여 한 번에 설치하세요.
```bash
pip install pandas xlrd streamlit plotly wordcloud matplotlib
```
*(참고: 집 PC에서는 굳이 가상환경(venv)을 새로 만들지 않고 글로벌 환경에 바로 설치하셔도 무방합니다.)*

### Step 3: 대시보드 서버 구동
라이브러리 설치가 끝났다면, 같은 폴더 위치의 터미널에서 다음 명령어를 실행합니다.
```bash
streamlit run app.py
```
명령어를 입력하면 자동으로 웹 브라우저가 열리며 대시보드가 나타납니다. (자동으로 열리지 않으면 창에 뜨는 `http://localhost:8501` 주소를 복사하여 브라우저에 붙여넣습니다.)

## 3. 구현 내용 및 데이터 처리 방식 요약 (What We Did)
현재 `app.py`에 구현되어 적용된 핵심 로직과 계획은 다음과 같습니다.

### 3-1. 철저한 오류 방지 파이프라인
1. **결측치 및 타입 에러 방어:** 
   발행연도에 문자열이 섞이거나 비어있을 경우 `NaN`으로 변환 후 제외하고, 인용된 총 횟수가 없으면 `0`으로 치환해 차트 그리기 단계에서 튕기는 현상을 막았습니다.
2. **KCI 형 변환 고려:** 
   내부가 HTML 태그로 얽혀 있거나 호환성을 보장하기 까다로운 구형 엑셀을 `xlrd` 라이브러리로 안전하게 읽고, 두 개의 데이터를 매끄럽게 하나로 `concat`하여 로드 속도를 최적화했습니다.

### 3-2. 고도화된 텍스트 마이닝 (키워드 추출)
- "연구", "분석", "고찰" 등 빈도수만 차지하는 보편적 조사 및 서술어(Stopwords)를 정규식으로 배제했습니다.
- 이를 통해 '살', '지각', '신체'와 같은 철학적 '개념어' 중심의 워드클라우드(Concept Map) 구성에 성공했습니다.

### 3-3. 대시보드 컴포넌트 구성 (UI/UX)
- 전역 폰트를 `Malgun Gothic`(맑은 고딕)으로 고정하여 모든 차트에서 한글 깨짐을 방지.
- **[총 4개의 탭 구현]**
  - **Tab 1: Total Insight** - 이중축(Bar+Line)으로 연표 및 전체 영향력 조망.
  - **Tab 2: Concept Map** - 핵심 15개 한국어 개념어와 워드클라우드 생성.
  - **Tab 3: People & Journal** - 주요 영향력 연구자 Top10과 주요 10대 학술지 비중 파이차트화.
  - **Tab 4: Research Library** - 초록과 메타데이터를 검색·정렬할 수 있는 데이터 테이블.

---

> **Tip:** `c:\dashboard` 폴더 전체를 압축하여 USB나 이메일에 담아서 집에 가져가시면 됩니다. `app.py` 안에는 파일 경로를 모두 안전하게 지정해 두었기 때문에 집에서도 동일한 `c:\dashboard` 위치에 파일을 두시면 더욱 깔끔하게 작동합니다!
