## 🛠 2. 오늘 수행한 작업 내역 (What We Did Today)

### (1) KCI Open API 기반 실시간 고품질 데이터 연동
- 기존의 정적인 엑셀 파일 데이터에서 벗어나, 발급받은 KCI Open API Key를 활용해 400여 편의 논문 상세 데이터(초록, 피인용 수 등)를 직접 추출하는 파이썬 스크립트(`fetch_and_preprocess_kci.py`)를 가동했다.
- 수집된 원시 데이터를 대시보드가 읽을 수 있도록 변환(`convert_csv_to_js.py`)하여, 분석 데이터의 질적 수준을 획기적으로 향상시켰다.

### (2) 대시보드 군집 분석(Clustering) 기능의 입체적 고도화
- **스트림그래프(Tab 4)**: 개별 키워드 빈도와 '패러다임 군집(신체·지각 vs 예술·언어)'을 전환하며 볼 수 있는 라디오 토글 버튼을 추가했다.
- **계층적 트리맵(Tab 2)**: 단순히 상위 키워드를 나열하던 것을 넘어, 전체 담론 중에서 각 패러다임이 차지하는 비중과 세부 키워드의 위계를 한눈에 보여주는 Treemap 차트를 새롭게 도입했다. 연도 슬라이더와 실시간으로 연동되어 역동적인 분석이 가능하다.

### (3) 데이터 정제(Data Cleaning)의 완벽화
- **AI 요약 부재 해결**: API 데이터에 누락되어 있던 요약본을 대체하기 위해, 초록 원문의 앞 3문장을 자동으로 추출하여 `초록_요약`을 렌더링하도록 수정했다.
- **동의어 병합 및 영단어 필터링**: "모리스 메를로-퐁티", "메를로--퐁티", "merleau-ponty" 등의 파편화된 동의어들을 모두 **"메를로-퐁티"** 하나로 강제 통합했다. 또한 Concept Map에서는 영단어를 배제하여 순수 한국어 철학 개념어들의 위계를 선명하게 살려냈다.

### (4) 가설 검증 및 기말 보고서 뼈대 구축
