## 🛠️ 세부 작업 계획 (To-Do List)

### 1. 🌐 `index.html` 데이터 소스 연동 및 업데이트
- **목표**: 새롭게 생성된 `퐁티_전처리완료.csv` (또는 이를 JSON으로 변환한 데이터)를 `index.html`에서 직접 불러오도록 수정합니다.
- **작업 내용**:
  - `PapaParse` 등을 활용한 클라이언트 사이드 CSV 파싱 혹은 데이터 내장(Embedding) 방식 적용.
  - 데이터 스키마 변경(`발행년도`, `인용수`, `학술지명` 등)에 맞게 기존 차트(Plotly) 및 워드클라우드 렌더링 로직 수정.
  - 새롭게 추가된 '초록' 데이터를 활용하여 상세 검색 및 요약 팝업 UI 고도화.

### 2. 🕸️ Concept Map (개념 네트워크) 클러스터링 시각화
- **목표**: "신체가 나오면 지각이 나오고, 언어가 나오면 공간이 나오는" 메를로-퐁티의 '연쇄 반응'을 증명합니다.
- **작업 내용**:
  - **Python 전처리 고도화**: 키워드 동시 출현 빈도(Co-occurrence Matrix)를 계산하여 노드(Node, 키워드)와 엣지(Edge, 연관성 강도) 데이터를 생성합니다 (`networkx` 활용).
  - **HTML/JS 시각화**: `Vis.js`, `D3.js`, 또는 `Plotly Network Graph`를 `index.html`에 도입하여 사용자가 노드를 클릭하고 드래그하며 얽힘(Entrelacs)을 탐색할 수 있는 인터랙티브 네트워크 맵을 구축합니다.

### 3. ⏳ 시대별 담론 흐름 (Time-Series Discourse Flow) 분석
- **목표**: '순수 현상학'에서 '디지털/응용 현상학'으로 이어지는 담론의 영토 확장을 시각적으로 보여줍니다.
- **작업 내용**:
  - `noUiSlider` 등의 타임라인 슬라이더를 `index.html` 사이드바에 연동.
  - 연도 슬라이더를 움직일 때마다 Concept Map의 네트워크 군집(Cluster)이 어떻게 변화하고 이동하는지 동적으로 렌더링(Animation).
  - 특정 시기(예: 2010년대)에 두드러진 긍정적 '익명성' 등의 특이 키워드 노드 강조.
