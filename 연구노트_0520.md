# 📝 연구노트 (2026년 5월 20일)

## 📌 1. 오늘의 작업 목표 및 기말과제 지향점
오늘의 핵심 과제는 기말과제의 요구사항인 **"나만의 관점과 문제의식이 담긴 분석적 대시보드"**를 향한 기술적/데이터적 기반을 완성하는 것이었다. 단순한 데이터 나열에 그쳤던 중간과제와 달리, 기말과제는 특정 '초점(Focus)'을 관통해야 한다. 
따라서 오늘 작업은 **"메를로-퐁티 담론의 표면적 키워드 너머, 시대별 철학적 관심사의 실질적 지형도(Topography) 변화를 어떻게 시각화할 것인가?"**라는 문제의식에 집중했다.

## 🛠 2. 오늘 수행한 작업 내역 (What I Did Today)

### (1) 서버리스(Serverless) 동적 데이터 파이프라인 구축
- 정제된 최종 데이터셋(`퐁티_엑셀_통합_전처리.csv`)을 브라우저 보안 제약(CORS) 없이 로컬과 GitHub Pages에서 즉각적으로 구동할 수 있도록 `data.js`로 변환하는 파이썬 자동화 스크립트를 작성 및 실행했다.
- 이로써 대시보드의 정적(Static) 한계를 극복하고, 언제든 데이터를 업데이트할 수 있는 실험적 환경을 마련했다.

### (2) '시대별 담론 흐름(Streamgraph)'의 분석적 고도화
- **도메인 지식을 활용한 데이터 필터링(AI와 인간의 협업)**: 단순히 빈도수가 높은 데이터를 그대로 보여주는 것은 의미가 없다는 판단 하에, "메를로-퐁티", "현상학", "모리스"와 같이 거시적이고 당연한 불용어(Stopwords)를 코드 단에서 필터링했다. 
- 그 결과, 거대 키워드에 가려져 있던 실질적 담론들(**신체, 지각, 예술, 언어, 타자 등**)의 시대별 부침(Rise and Fall)이 선명하게 드러났다.
- **시각적 정밀도 향상**: 결측 연도를 0으로 보간(Padding)하고, 딱딱한 직선 대신 곡선(Spline) 보간법을 적용하여 개념들이 역사 속에서 어떻게 교차(Chiasm)하고 진화해왔는지를 부드러운 물결 형태로 구현했다.
